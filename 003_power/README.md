# Power
Version osf.010.v0.2

Use the 'host' connector on the power supply side.
## pinning

| Pin | Description |
| --- | ----------- |
| 1   | GND         |
| 2   | GND |
| 3   | GND |
| 4   | GND |
| 5   | GND |
| 6   | 1.2V |
| 7   | 1.8V |
| 8   | 2.5V |
| 9   | 3.3V |
| 10  | 5V   |


 Power connectors should be oriented on the top (device) or bottom (host) of the board.
