# 1000BASE-T
## pinning
| Pin | Description |
| --- | ----------- |
| 1   | GND  |
| 2   | TD1+ |
| 3   | TD1- |
| 4   | TD2+ |
| 5   | TD2- |
| 6   | GND  |
| 7   | TD3+ |
| 8   | TD3- |
| 9   | TD4+ |
| 10  | TD4- |

this connector should be oriented on the top (magnetics side) or the bottom (phy side) of a board.
